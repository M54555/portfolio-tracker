Come attivare la sincronizzazione Firebase
Passo 1 — Vai su console.firebase.google.com e crea un nuovo progetto (es. "portfolio-tracker").
Passo 2 — Nel progetto, clicca Aggiungi app → Web e dai un nome all'app. Copia l'oggetto firebaseConfig.
Passo 3 — Nel menu laterale vai su Authentication → Metodi di accesso e abilita Google.
Passo 4 — Vai su Firestore Database → Crea database → scegli Modalità produzione e la regione più vicina.
Passo 5 — In Firestore → Regole, incolla queste regole e pubblica: 
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /users/{userId}/data/{doc} {
      allow read, write: if request.auth != null
                         && request.auth.uid == userId;
    }
  }
}
Passo 6 — Apri questo file HTML in un editor di testo, trova il blocco firebaseConfig (in cima allo script) e sostituisci i valori con quelli copiati al Passo 2. Salva e riapri il file.

⚠ Nota importante: Per usare il login Google, il file deve essere servito da un server web (non aperto direttamente come file locale). Usa Netlify Drop — trascina il file HTML e ottieni un URL pubblico gratuito in 30 secondi.
